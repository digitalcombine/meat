\subsection{Numeric Class}

The Numeric Class is an abstract base class for all numeric or mathematical values.

\begin{lstlisting}
Object subClass: Numeric body: {
  class method minValue
  class method maxValue
  method + other
  method - other
  method * other
  method / other
  method % other
  method ^ other
  method == other
  method != other
  method < other
  method <= other
  method > other
  method >= other
  method negative
  method absolute
  method compliment
  method asText
}
\end{lstlisting}
